About

Impulses allows the users to set up custom rule-based reminders to save rather than spend money on that purchase. It will send the user a text at a time specified by the user to suggest saving that money rather than spending it. Users are able to make the deposit of the savings via a simple yes/no response to the text message and the “coffee money” would be automatically drawn from the user’s account and deposited into a savings/investment account.
